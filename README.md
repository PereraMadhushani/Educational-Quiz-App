# Science Quiz Master

A Flutter mobile application for interactive science quizzes.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK (included with Flutter)
- IDE: Android Studio, VS Code, or IntelliJ IDEA

### Project Setup

1. **Install Flutter**
   - Download from [flutter.dev](https://flutter.dev)
   - Follow the installation guide for your OS

2. **Get Dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the Application**
   ```bash
   flutter run
   ```

### Project Structure

```
lib/
├── main.dart          # Entry point of the application
test/
├── widget_test.dart   # Widget tests
assets/               # Images, fonts, and other assets
```

### Available Commands

- `flutter run` - Run the application on connected device/emulator
- `flutter test` - Run all tests
- `flutter analyze` - Analyze code for issues
- `flutter build apk` - Build APK for Android
- `flutter build ios` - Build for iOS

## Features

- Interactive quiz interface
- Science-based questions
- Real-time scoring
- User-friendly UI

## Development

To add new features or modify the app:

1. Edit files in the `lib/` directory
2. Hot reload with `r` command in terminal while app is running
3. Run tests: `flutter test`

## Testing

Run tests with:
```bash
flutter test
```
